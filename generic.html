Case1

This case is taken from a real freelance project that I worked on.
the dataset isn't real data. It is a newly created dataset that is similar in nature and can be used comfortably.

Requirements from customer :

1. A graph showing daily sales for each branch.
2. A graph showing daily total sales for all branches.
3. A pie chart showing the distribution of sales for each branch combined.
4. A graph showing the average sales for each branch.
5. The store with the highest/lowest sales.
6. The number of stores for each branch.

Report information

The report consists of 2 pages. The first page is an Overview page that contains most of the requirements related to sales information. The main focus of this page is the branch performance and sales insights of the company.
The second page is the Customer page which displays the details of the branches/customers, such as the total number of shops in each branch or the total sales of every customer.
Additionally, I have included a heatmap chart that provides a clear view of sales performance throughout the day.
